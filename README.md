EverPie Pizza Sales Analysis is an SQL project analyzing pizza orders data. It includes queries to explore daily/ hourly order trends, top-selling pizzas, category-wise distribution, revenue calculation, and cumulative revenue using window functions.

>>> Key Insights Covered
Daily and hourly order distribution
Most ordered pizza sizes
Category-wise sales performance
Top 3 pizzas by quantity and by revenue
Revenue analysis using joins
Cumulative revenue trends
Window functions (RANK, DENSE_RANK, ROW_NUMBER, OVER)

>>>  Skills Used
SQL Joins
GROUP BY + Aggregations (SUM, COUNT, AVG)
Window Functions
ORDER BY & LIMIT
Data Cleaning & Formatting

>>>  Project Goal

To understand sales trends, customer ordering behavior, and revenue-driving pizza types through database queries.

I learned and practiced SQL using tutorials from the WS Cube Tech YouTube channel.
This pizza sales project was inspired by their lessons, and I extended it with additional analysis such as:

Revenue calculations
Ranking pizzas by categories
Window functions
Hourly and daily sales trends

>>>  This project helped me strengthen:

SQL joins

Aggregate functions (SUM, COUNT, AVG)
Window Functions (RANK, DENSE_RANK, OVER)
Business analytics thinking
